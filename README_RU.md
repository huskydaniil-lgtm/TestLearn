## Обзор

TestLearn — интерактивная учебная платформа, которая помогает пользователям освоить фундаментальные концепции тестирования программного обеспечения через структурированную теорию, практические викторины и всесторонний глоссарий терминов. Платформа реализует систему отслеживания прогресса на основе сессий для мониторинга продвижения пользователей по учебному материалу.

### Ключевые характеристики
- **Образовательная направленность**: Полное покрытие основ тестирования программного обеспечения
- **Интерактивное обучение**: Теория в сочетании с самопроверкой через викторины
- **Отслеживание прогресса**: Сессионный мониторинг изученных тем и результатов викторин
- **Адаптивный дизайн**: Корректное отображение на настольных и мобильных устройствах
- **Открытый исходный код**: Лицензия MIT для образовательного и коммерческого использования

## Функции

### Учебные модули
- 📚 **Структурированная теория**: 16 тем, организованных по 5 категориям тестирования
- 🔍 **Умный поиск**: Полнотекстовый поиск по теории и глоссарию
- 📖 **Глоссарий**: 43 основных термина тестировщика с фильтрацией по буквам
- 🏆 **Отслеживание прогресса**: Мониторинг изученных тем и результатов викторин
- 🔖 **Закладки**: Сохранение важных тем для быстрого доступа

### Инструменты оценки
- 📝 **Интерактивные викторины**: 6 тестов с 50+ вопросами всего
- ⏱️ **Сессии с таймером**: Опциональный таймер для реалистичности викторин
- 🔄 **Рандомизация вопросов**: Разный порядок вопросов при каждой попытке
- 💡 **Подробные объяснения**: Изучение материала через объяснения правильных и неправильных ответов
- 📊 **Аналитика результатов**: Просмотр истории успеваемости и экспорт в CSV

### Техническая реализация
- ⚡ **Бэкенд на FastAPI**: Высокопроизводительный фреймворк на Python
- 🗃️ **База данных SQLite**: Лёгкое, не требующее настройки хранилище
- 🎨 **Tailwind CSS**: Современный utility-first подход к стилизации (через CDN)
- 🔧 **Шаблонизатор Jinja2**: Чистое разделение представления и логики
- 📱 **Адаптивная вёрстка**: Мобильно-дружелюбный дизайн без фреймворков

## Проектирование базы данных

Платформа использует SQLite с 8 взаимосвязанными таблицами, построенными по принципам нормализации:

### Основные таблицы
1. **categories** - Виды тестирования (функциональное, нефункциональное, методологии и т.д.)
2. **topics** - Образовательные материалы, связанные с категориями
3. **quizzes** - Коллекции викторин (опционально связанные с категориями)
4. **questions** - Элементы викторин с вариантами ответов
5. **quiz_results** - Записи попыток пользователей с подсчётом баллов
6. **glossary** - Справочник терминов с алфавитической категоризацией
7. **feedback** - Отзывы и оценки пользователей
8. **user_progress** - Отслеживание прогресса обучения на основе сессий
9. **read_topics** - Связь многие-ко-многим: сессии ↔ темы (прогресс чтения)
10. **bookmarks** - Связь многие-ко-многим: сессии ↔ темы (закладки пользователей)

### Диаграмма сущностей и связей
Полную схему базы данных можно посмотреть в приложении по адресу `/database` или в файле `templates/database.html` (диаграмма Mermaid.js).

## API endpoints

TestLearn предоставляет RESTful API для программного доступа:

| Endpoint | Метод | Описание | Ответ |
|----------|-------|----------|-------|
| `/api/stats` | GET | Статистика платформы в целом | JSON |
| `/api/categories` | GET | Список всех категорий тестирования | JSON массив |
| `/api/quiz/{id}/questions` | GET | Вопросы конкретной викторины (без ответов) | JSON массив |
| `/api/feedback` | GET | Недавние отзывы пользователей | JSON массив |

## Начало работы

### Требования
- Python 3.9 или выше
- Менеджер пакетов pip
- Git (опционально, для клонирования репозитория)

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/QuadDarv1ne/TestLearn.git
   cd TestLearn
   ```

2. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   # Активация окружения:
   # Windows:
   venv\Scripts\activate
   # Linux/macOS:
   source venv/bin/activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

### Запуск приложения

#### Прямой запуск
```bash
# Запуск сервера разработки
uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# Открытие в браузере
http://localhost:8000
```

#### Использование Docker
```bash
# Сборка и запуск с Docker Compose (рекомендуется)
docker-compose up --build

# Приложение будет доступно по адресу http://localhost:8000

# Для запуска в фоновом режиме:
docker-compose up -d

# Для остановки и удаления контейнеров:
docker-compose down
```

#### Ручное использование Docker
```bash
# Сборка Docker-образа
docker build -t testlearn .

# Запуск контейнера
docker run -p 8000:8000 -v $(pwd)/testlearn.db:/app/testlearn.db testlearn
```

Для развертывания в продакшене рекомендуется использовать сервер ASGI уровня production, например, Hypercorn или рабочие процессы Uvicorn за обратным прокси.

## Использование

### Навигация по платформе
1. **Главная страница**: Начните с целевой страницы для получения общего представления
2. **Раздел теории**: Просматривайте или ищите учебные темы по категориям
3. **Темы**: Нажмите на любую тему, чтобы прочитать подробное содержание с примерами
4. **Викторины**: Проверьте свои знания в разделе викторин
5. **Глоссарий**: Ищите термины тестирования в любой момент
6. **Статистика**: Просматривайте свой учебный прогресс и экспортируйте результаты
7. **Обратная связь**: Делитесь своими мыслями, чтобы помочь улучшить платформу

### Административные функции
Доступ к админ-панели по адресу `/admin` (обратите внимание: сейчас отсутствует аутентификация - только для учебных целей):
- Управление категориями, темами, вопросами и терминами глоссария
- Просмотр системной статистики и обратной связи пользователей
