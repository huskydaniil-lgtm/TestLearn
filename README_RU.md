# TestLearn — Учебная платформа по тестированию ПО

Образовательный веб-сайт для изучения основ тестирования программного обеспечения.
Курсовой проект по дисциплине «Базы данных» МФЮА.

## Технологии

- **Backend:** FastAPI (Python)
- **База данных:** SQLite (реляционная)
- **Шаблоны:** Jinja2
- **Фронтенд:** HTML + Tailwind CSS (CDN) + Vanilla JS
- **Стилизация:** Tailwind CSS с кастомной цветовой схемой (emerald/green)

## Структура проекта

```
fastapi-app/
├── main.py              # Основное приложение FastAPI
├── templates/           # Jinja2 шаблоны
│   ├── base.html        # Базовый шаблон (навигация, подвал, блоки)
│   ├── index.html       # Главная страница
│   ├── theory.html      # Теоретический материал
│   ├── topic.html       # Страница отдельной темы
│   ├── quiz_list.html   # Список тестов
│   ├── quiz.html        # Прохождение теста
│   ├── quiz_result.html # Результаты теста
│   ├── glossary.html    # Глоссарий терминов
│   ├── stats.html       # Статистика
│   ├── database.html    # Структура базы данных (с ER-диаграммой)
│   └── feedback.html    # Обратная связь

```

## База данных

СУБД: SQLite с 8 таблицами:

| Таблица | Описание |
|---------|----------|
| `categories` | Категории (виды тестирования) |
| `topics` | Теоретические темы и уроки |
| `quizzes` | Тесты для проверки знаний |
| `questions` | Вопросы тестов (с вариантами ответов) |
| `quiz_results` | Результаты прохождения тестов |
| `glossary` | Словарь терминов тестировщика |
| `feedback` | Сообщения обратной связи |
| `user_progress` | Прогресс пользователя |

## Установка и запуск

### 1. Клонировать репозиторий

```bash
cd fastapi-app
```

### 2. Создать виртуальное окружение (рекомендуется)

```bash
python -m venv venv
# Linux/macOS:
source venv/bin/activate
# Windows:
venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить сервер

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 5. Открыть в браузере

```
http://localhost:8000
```

## Функционал

- 📚 **Теория** — 16 тем по 5 категориям тестирования ПО, поиск по темам
- 📝 **Тесты** — 6 тестов (50 вопросов) с таймером, рандомизацией и объяснениями ответов
- 📖 **Глоссарий** — 43 термина с поиском и фильтрацией по буквам
- 📊 **Статистика** — аналитика прохождения тестов, экспорт в CSV, сброс прогресса
- 🗄️ **База данных** — структура БД с ER-диаграммой (Mermaid.js)
- 💬 **Обратная связь** — форма для отправки сообщений, список последних отзывов
- 📱 **Адаптивный дизайн** — корректное отображение на мобильных устройствах
- 🔍 **Поиск** — поиск по теории и глоссарию

## API эндпоинты

- `GET /api/stats` — статистика платформы (JSON)
- `GET /api/categories` — список категорий (JSON)
- `GET /api/quiz/{id}/questions` — вопросы теста без ответов (JSON)
- `GET /api/feedback` — список отзывов (JSON)
